# deyunio/smwds
自动化运维平台 based on SaltStack

#项目情况
* WEB框架：Flask
* 配置管理：Saltstack 
* 监控集成：Sensu
* 虚拟网络: Calico
* CMDB: Posgresql
* 任务调度: Mesos
